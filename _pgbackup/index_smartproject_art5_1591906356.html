Smart competentie(s): Smart Projects & Smart Start

Eindverslag


Namens de projectgroep heb ik de documenten overhandigd aan onze docenten van de minor. Het aangeleverde rapport is een onderzoeksrapport met een werkend prototype. Het prototype zal apart aangeleverd worden via een map (zip).

De studenten van de minor Smart Industry hebben aan de hoofdvraag voldaan door allereerst Node-Red op een RPI en een cloud instantie te installeren. Nadat dit gelukt is, zal er een script op een RPI geïnstalleerd moeten worden. Deze zorgt ervoor dat via “one-way” synchronisatie de Node-Red cloud configuratie overgenomen wordt op de Node-Red RPI-configuratie.

Doordat het script op elk apparaat kan werken die bestaat uit een Raspberry Pi architectuur, kan naast een RPI ook bijvoorbeeld een UniPi (industriële RPI) vanuit de cloud aangestuurd worden.

De infographic geeft een duidelijke illustratie van het prototype voor Icr3ate.
