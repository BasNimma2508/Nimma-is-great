Mijn eerste eigen website

Smart competentie(s): Smart connection & Smart Business


In het begin vond ik het uitdagend om een eigen website te bouwen in Microsoft visual studio code, voornamelijk met de programmeertalen HTML (HyperText Markup Language) en CSS (Cascading Style Sheets).

In het begin had ik website/ portfolio gemaakt in studio Visual studio code. Alleen vond ik het design en de navigatie erg storend. Hierdoor besloot ik om eventuele hulpmiddelen te gebruiken voor mijn website.

Ik ben gewend aan stijlvolle designs en navigatie van de Wix tools. Hierom heb ik besloten om zo ver mogelijk te komen met een stijlvol design en gebruiksvriendelijke navigatie in mijn website. Ik heb een template aangeschaft via pixelarity.com, hier kun je stijlvolle template downloaden en gebruiken voor je eigen website.

Deze templates geven alleen een zetje om te beginnen met je website. In deze template zijn enkele bestanden verwerkt, zodat de website er stijlvoller uitziet. Toch moet je zelf veel verwerken in MS Visual Studio Code, zodat de website mijn eigen verwachtingen krijgt.

Ik merk met mijn bedrijfskundige achtergrond dat een website maken uitdagend kan zijn, maar uiteindelijk kom je er wel. Door middel van ondersteuning van het web, medestudenten en doorzettingsvermogen heb ik een redelijk kunnen neerzetten. Ik hoop dat iedereen ook van mijn website kan genieten.
