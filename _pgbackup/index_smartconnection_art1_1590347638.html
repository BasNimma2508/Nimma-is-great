Casus: Hoe blijven kleine ondernemers zonder technische kennis actief in de onlinewereld?


Ik heb voor kleine ondernemers een website gemaakt, afgestemd op ieders individuele wensen. Deze cliënten hebben weinig tot geen ervaring met computers, daarom heb ik besloten om deze ondernemers te helpen. Ik heb voor elke ondernemer een website en abonnementen afgesloten via Wix. Voor degene die niet weten wat Wix.com is, hier staat een uitgebreide uitleg met een bijbehorende link.


“Wat is Wix? Wanneer je weinig ervaring hebt met het ontwikkelen van websites of webwinkels, kan het gebruik van Wix het overwegen waard zijn. Wix is een handige online tool, waarmee je binnen een paar uur een complete website in elkaar klikt! Een van de voornaamste voordelen van Wix, is het feit dat je geen programmeerkennis nodig hebt om met je website aan de slag te gaan. Mocht je een stapje verder willen gaan, dan kun je ook eens kijken naar Wordpress. Wordpress biedt net iets meer mogelijkheden, maar is daardoor ook lastiger te gebruiken dan Wix.”

Hoe maak ik een website voor deze ondernemers? Ik heb aandachtig geluisterd voor zijn specifieke wensen en verwachtingen van de website. Vervolgens heb ik ook een handleiding aan elke ondernemer verstuurd. Hierdoor krijgt elke cliënt de kans om in de toekomst zelf specifieke elementen, foto’s, video’s, evenementen, beurzen, en producten toe te voegen aan zijn/haar eigen website.

Waarom hiep ik deze ondernemers? Tegenwoordig is het redelijk prijzig om sites te onderhouden door een derde persoon, maar zelf een site onderhouden zonder enige technische kennis is ook belemmerend. Daarom heb ik deze ondernemers geholpen met hun website op het gebied van inrichting, navigatie, design en overige features, oftewel een kant-en-klare website template.

Vervolgens kan ieder individu zelf zijn/haar website onderhouden in de toekomst, zonder enig tussenkomst van een derde persoon.

Ik heb aandachtig geluisterd voor ieders specifieke wensen en verwachtingen van hun/haar website. Vervolgens heb ik ook een handleiding aan elke ondernemer verstuurd. Indien hulp wel gewenst is, dan kan elke ondernemer een aparte afspraak maken, zodat deze handeling mogelijk wel ingericht kan worden.
